# homebrew-vex

Official optional Homebrew tap for [vex](https://github.com/imnotnoahhh/vex).

`vex` keeps direct installation as the primary path, but this tap exists for users who already manage developer tools with Homebrew.

## Install

```bash
brew install imnotnoahhh/homebrew-vex/vex
```

## Verify

```bash
vex --version
vex init --dry-run
```

## Updating

The formula is generated from published GitHub release assets and is intended to be updated automatically by the main `vex` repository's release postflight workflow.
